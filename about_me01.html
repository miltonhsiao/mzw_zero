簡介

我是一位結合圖像創作與程式語言的實驗型創作者，致力於以數位方式表達自然結構、流動與時間感。 我的創作常融合生成藝術（Generative Art）、互動設計與感知體驗。

過去我學習了網頁前端、創意編碼（p5.js / Three.js / GLSL），也持續研究構圖、美感與視覺語彙。透過寫程式，我找到了對世界的新觀看方式。

創作理念

在我看來，創作不只是展示技能，更是一種探索——對內探索自己如何感知世界，對外探索程式與視覺之間的詩性可能。 我的作品往往從一個概念、一段錯誤的程式碼，甚至一段模糊記憶出發，慢慢發展成視覺生命體。

工具與媒材

常用技術：HTML / CSS / JavaScript、p5.js、Three.js、GLSL、Processing
視覺創作：生成圖像、動態構圖、視覺程式化結構
