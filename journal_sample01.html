從混沌到構圖：一次生成圖像的靈感之旅

2025 年 5 月 18 日 ・ #generative-art #p5js

有時候創作並非來自明確的計畫，而是一段段的失敗與意外的交錯。

這次我從一個色塊亂數排列的小實驗開始，原本只是想做一個「隨機色彩構成的方格牆」，但幾次不經意的參數錯誤，讓我看到了「破壞結構」的美感。

下方是當時的核心程式碼片段：

for (let i = 0; i < cols; i++) {
  for (let j = 0; j < rows; j++) {
    let w = width / cols;
    let h = height / rows;
    let c = color(random(255), random(255), random(255), 150);
    fill(c);
    rect(i * w, j * h, w, h);
  }
}

結論是：意外往往比規劃更誠實。當我不再執著完美構圖，而是觀察程式本身的「行為」，它反而回饋了我一種節奏。
